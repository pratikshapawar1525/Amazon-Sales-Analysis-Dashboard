# 📊 Amazon Sales Analysis Dashboard

## Project Objective

The Amazon Sales Analysis Dashboard was created to analyze sales performance and uncover valuable business insights. The dashboard helps stakeholders understand customer purchasing behavior, product performance, and revenue trends to support data-driven decision-making.

## Dataset Used

- <a href="https://github.com/pratikshapawar1525/Amazon-Sales-Analysis-Dashboard/blob/main/Amazon_Combined_Data.xlsx">Amazon_Combined_Data</a>

- <a href="https://github.com/pratikshapawar1525/Amazon-Sales-Analysis-Dashboard/blob/main/Amazon%20sales%20Analysis%20dashboard.pbix">Amazon sales Analysis dashboard</a>

## Questions (KPIs)

* Compare sales and revenue using a single chart.
* Which month recorded the highest sales?
* Which products generated the most revenue?
* What are the top-performing product categories?
* Which regions contributed the highest sales?
* What are the overall sales trends over time?
* Which products have the strongest sales performance?
* What insights can be derived from category-wise analysis?

## Process

* Verified the dataset for missing values and inconsistencies.
* Cleaned and transformed data using Power Query.
* Created calculated measures and KPIs using DAX.
* Built interactive visualizations to answer business questions.
* Added slicers and filters for dynamic reporting and analysis.
* Designed a user-friendly dashboard for easy data exploration.

## Dashboard Features

* 📈 Sales Trend Analysis
* 💰 Revenue Performance Tracking
* 🛍️ Product Performance Analysis
* 📂 Category-wise Insights
* 🌍 Regional Sales Analysis
* 🎯 Interactive Filters & Slicers
* 📊 KPI Monitoring
